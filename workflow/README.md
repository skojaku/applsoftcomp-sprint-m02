Place your Python scripts for data processing and visualization here.

For example:
- `viz_1d_data.py` — Script to load, process, and visualize `1d-data.csv`
- `viz_2d_data.py` — Script to load, process, and visualize `2d-data.csv`
- `viz_1d_multi_method.py` — Script to load, process, and visualize `1d-multi-method-data.csv`

Each script should save its output figure to `../figs/`.
